# Simple Responsive Website

This is a simple responsive website project created for the Software Development Skills Front-End module. The website includes navigation, responsive CSS, Flexbox, CSS Grid, cards, FAQ section, mobile menu, and a simple parallax-style section.

## Project Files

After extracting the ZIP file, you should see files similar to these:

```
simple-responsive-site/
│
├── index.html
├── style.css
└── README.md
```

## How to Run the Project

### Option 1: Open Directly in Browser

1. Download the project ZIP file.
2. Right-click the ZIP file and choose **Extract All**.
3. Open the extracted project folder.
4. Find the file named **index.html**.
5. Double-click **index.html**.
6. The website will open in your default browser.

### Option 2: Open with a Specific Browser

1. Open the extracted project folder.
2. Right-click **index.html**.
3. Choose **Open with**.
4. Select **Google Chrome**, **Microsoft Edge**, or another browser.

## How to Edit the Project

1. Open the project folder in **Visual Studio Code** or another code editor.
2. Edit **index.html** to change the website content.
3. Edit **style.css** to change the design, colors, layout, and responsiveness.
4. Save the files.
5. Refresh the browser to see the changes.

## How to Deploy with GitHub Pages

1. Create a new repository on GitHub.
2. Upload the project files: `index.html`, `style.css`, and this `README.md` file.
3. Go to the repository **Settings**.
4. Open the **Pages** section.
5. Under **Branch**, select `main` and `/root`.
6. Click **Save**.
7. GitHub will generate a live website link after a short time.

## Technologies Used

- HTML
- CSS
- Flexbox
- CSS Grid
- Responsive Design
- GitHub Pages for deployment

## Notes

This project does not require Node.js, npm, MongoDB, Express, or React to run. It is a simple static front-end website, so it can be opened directly in a browser.
