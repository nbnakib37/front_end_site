# Digital Service Shop

A simple responsive course project website based on the Front-End learning diary activities.

## Project requirements covered

1. Page with navigation
2. Responsive styles with CSS
3. Flexbox and CSS Grid
4. Ready for deployment with GitHub Pages

## Main features

- Responsive navigation with mobile menu
- Hero section
- Service cards using CSS Grid
- Process section using Flexbox
- Parallax-style banner
- FAQ accordion using HTML details/summary
- Contact section and footer

## How to run

Open `index.html` in a browser.

## How to deploy with GitHub Pages

1. Create a new GitHub repository.
2. Upload these project files.
3. Go to Settings > Pages.
4. Select the main branch and root folder.
5. Save and wait for GitHub to publish the live website link.
